# O Problema Não É o Seu Funil

Landing page estática sobre a metodologia de estruturação de funis de negócios.

## Stack

- HTML / CSS / JavaScript (estático)
- Hospedagem: Vercel

## Como executar

Apenas abra `index.html` no navegador ou sirva com um servidor local:

```bash
python3 -m http.server 8000
```

Acesse em `http://localhost:8000`

## Estrutura

- `/index.html` — Landing page principal
- `/css/` — Estilos
- `/js/` — Scripts
- `/img/` — Imagens e assets
- `/favicon.svg` / `/favicon.png` — Favicon

## Deploy

O projeto é deployado automaticamente na Vercel quando há push para `main`.
